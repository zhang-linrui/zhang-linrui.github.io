安装tensorrt流程

  • 首先需要安装好NVIDIA驱动、cuda、cudnn，搞清楚cuda版本，cudnn版本。

  • 安装好上面步骤之后，到这里https://developer.nvidia.com/nvidia-tensorrt-download 下载tensorRT 7.0的安装包，注意要选对符合你cuda和cudnn的对应版本的tar file installed packages。一定要选tar的！
